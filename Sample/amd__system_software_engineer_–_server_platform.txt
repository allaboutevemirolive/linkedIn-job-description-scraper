System Software Engineer – Server Platform

The Role

AMD is looking for a highly talented engineer to join our Datacenter Solution Engineering team as an overall expert in Server system software Engineer. System Software Engineer will work closely with our customers who are developing AMD EPYC based systems. This Engineer will develop end-to-end customer debug tool solution which help team to resolve customer critical issues in time through the debug tool time to meet product development or sustaining milestones. Opportunity to learn emulation and enable debug tools for post-silicon used.

The Person

A successful candidate is one who can drive technical decisions independently, has accountability for accuracy, reliability and completeness of assignment results critical to the project’s success. This role allows the opportunity to influence technical decisions that have a significant impact on multiple products or the product line. 

Key Responsibilities

 * Have the strong python software knowledge and programming skills which can handle large scale coding, hardware\driver interactions with software stacks. Familiar with software API handshake to HAL (Hardware Abstraction Layer) frameworks.
 * Develop customer debug tools for server industry standard buses, such as DDR, PCIe, SATA, and other high speed IO protocol.
 * Deliver the end-to-end solution to customer which including define tools framework, coding, tools validation, enable the tools installation package to customer.
 * Provide the customer debug tool support and root cause analysis, knowing how to design the experiment by rule out, root cause the issue by co-work with INT\EXT team well.
 * Further requirements include a strong understanding of BIOS, OS, BMC, and driver interactions at the system level as well as proficient understanding of x86 CPU architecture and functionality.
 * A self-starter required and able to deal with a high level of ambiguity.
   
   

Preferred Experience

 * Engineering degree in Electrical Engineering, Computer Engineering, or Computer Science with 12+ years of extensive experience.
 * The ideal candidate has worked at computer company as a developer who gone through the full software development product life cycle.
 * Worked on multiple projects from development, through bring-up and validation and supported through the release to production.
 * Strong knowledge of software architecture and OOP (Object Oriented Programming) desired.
 * Experience in large scale software coding, data visualization, machine learning practice and familiar libraries such as Keras\Tensorflow, OpenCV, NumPy\SciPy for models build.
 * Excellent communication skills are required in the role of supporting customers.
   
   

Academic Credentials

 * Bachelor’s or Master’s in computer engineering or computer science or electrical engineering, or comparable disciplines
